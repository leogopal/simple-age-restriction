== Description ==

Age Gate for WordPress

== Installation ==

1. Upload the 'simple-age-restriction' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Visit 'Settings > Simple Age Restriction' and adjust your configuration.
